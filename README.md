> A professional-grade deep learning library featuring **from-scratch implementations** (NumPy) and **state-of-the-art architectures** (PyTorch) across Computer Vision, NLP, and Generative AI.

---

## ✨ Key Features

- 🏗️ **From-Scratch Foundations** — MLP, Optimizers (Adam), and Attention implemented in pure NumPy.
- 📸 **Computer Vision** — ResNet architectures and CNN projects (Face Recognition, VGG16).
- 🧠 **NLP & Sequences** — Scaled Dot-Product Attention and RNN/LSTM modules.
- 🎨 **Generative Models** — Variational Autoencoders (VAE) and GAN foundations.
- 🛠️ **Production Best Practices** — Batch Norm, Dropout, and professional training loops.

---

## 📂 Project Structure

